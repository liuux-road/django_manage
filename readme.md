### 1. 新建 **Django** 项目

### 2. 连接git

```shell
$ git init
$ git add README.md
$ git commit -m "first commit"
$ git branch -M main
$ git remote add origin git@github.com:liuux-road/my_first_WEBpage.git
$ git push -u origin main
```
